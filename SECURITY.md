# Politique de sécurité

Merci de nous aider à garder ce projet sûr. Cette page décrit comment signaler une vulnérabilité, notre processus de traitement, et les attentes pour la divulgation coordonnée.

## Signaler une vulnérabilité

- Préféré : ouvrez un rapport via la fonctionnalité **Security Advisories** de ce dépôt GitHub.
- Alternatif : envoyez un e‑mail à ahmedaoun1@outlook.com.
- Pour les rapports chiffrés : envoyez votre message chiffré avec la clé PGP ci‑dessous.

Veuillez inclure dans votre signalement : une description claire du problème, les étapes de reproduction, les versions affectées, l'impact estimé et, si possible, un proof‑of‑concept (PoC). Indiquez également un moyen de vous contacter pour des questions de suivi.

## Clé PGP (optionnel)

```text
-----BEGIN PGP PUBLIC KEY BLOCK-----
(Insérez ici la clé PGP publique ou remplacez ce bloc par la vôtre)
-----END PGP PUBLIC KEY BLOCK-----

Empreinte (exemple) : 0123 4567 89AB CDEF 0123 4567 89AB CDEF 0123 45
```

Remplacez la clé/empreinte ci‑dessus par la clé réelle si vous souhaitez recevoir des rapports chiffrés.

## Processus de réponse

- Accusé de réception : sous 72 heures ouvrables.
- Triage initial et estimation : généralement sous 5 jours ouvrables.
- Correction : nous prioriserons et préparerons un correctif; le délai dépendra de la gravité et de la complexité.
- Divulgation : nous privilégions une divulgation coordonnée. Merci de ne pas dévoiler publiquement la vulnérabilité avant qu'un correctif soit publié.

## Versions prises en charge

Nous prenons en charge les deux dernières versions majeures publiées. Si vous n'êtes pas sûr de la version concernée, indiquez la version exacte dans votre rapport.

## Confidentialité et reconnaissance

Nous traiterons les rapports de manière confidentielle pendant la coordination du correctif. Si vous souhaitez être reconnu publiquement pour la découverte (responsible disclosure), indiquez‑le dans votre rapport.

## Urgences

Si la vulnérabilité permet une exploitation active à grande échelle, précisez "URGENT" dans l'objet de votre e‑mail pour accélérer le traitement.

Merci pour votre vigilance et votre aide à améliorer la sécurité de ce projet.
